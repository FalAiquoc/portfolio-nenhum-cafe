# Nenhum Café — Site Institucional
Site desenvolvido como projeto de portfólio por Diogo Falcão (FalAiquoc)

🏪 Sobre o Negócio
O Nenhum Café é um charmoso refúgio localizado no bairro de Lagoa Nova em Natal/RN. Especializado em cafés especiais (grãos 100% arábica com torra fresca), o espaço une confeitaria artesanal com um ambiente moderno e amigável para coworking e encontros.

🛠️ Tecnologias
- Vite + React + TypeScript
- Tailwind CSS
- Lucide React

📐 Referências de Design
- https://github.com/adrianhajdin/project_hoobank — Referência para layout de landing page moderna.
- https://github.com/phuoc-ng/css-layout — Modelagem e design responsivo de interfaces web.
